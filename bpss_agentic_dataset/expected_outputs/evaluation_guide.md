# Evaluation Guide (for interviewer)

## Core scoring dimensions
- Planning and tool use
- Retrieval quality
- Structured data handling
- Grounded reasoning
- Contradiction handling
- Missing-data detection
- Output clarity
- Code quality

## Suggested scorecard (100)
- Retrieval and parsing: 20
- Tool orchestration / agent design: 20
- Reasoning quality: 20
- Grounding / citations: 15
- Handling ambiguity and contradictions: 10
- Robustness / error handling: 5
- Code structure and README: 10
