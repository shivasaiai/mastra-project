# Sample Questions

1. Which candidate files are **not yet ready for BPSS closure**, and why?
2. For each candidate, determine whether the **identity**, **employment history**, **criminality**, and **right-to-work** checks are complete. Cite evidence.
3. Which cases appear to have **policy exceptions**? Distinguish approved exceptions from unapproved deviations.
4. Compare the adjudication note for **CAND-104** with the policy. Is closure justified?
5. Which candidates have potentially **stale documents** or evidence that expired before review completion?
6. Draft an escalation summary for the Screening Operations Lead covering the top 3 highest-risk cases.
7. What information is missing to determine whether **CAND-105** can be cleared?
8. Reconcile any contradictions between the tracker and the analyst notes.
9. Which candidates were marked "ready to join" even though one or more mandatory controls were incomplete?
10. Create a structured JSON summary for each candidate with:
   - status
   - control completion by category
   - missing items
   - risk level
   - supporting evidence
